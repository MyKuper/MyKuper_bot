# MyKuper Bot

Асинхронный бот для управления кланом в Clash of Clans с интеграцией PostgreSQL и официальным API.

## Возможности
- Умный план войны: авто-распределение целей, таблица атак, логирование
- Статистика CWL: отслеживание лиги, звёзд и состава группы
- Управление кланом: пожертвования, состав участников, информация о столице
- Авто-напоминания: уведомления об атаках за 4 часа и 1 час до конца войны
- Профиль игрока: привязка аккаунтов, личная статистика, история войн
- Экспорт отчётов: генерация .md-файлов по итогам войны

## Библиотеки
| Инструмент     | Назначение                     |
|----------------|--------------------------------|
| `Python 3.10+` | Язык программирования          |
| `aiogram 3.x`  | Асинхронный Telegram API       |
| `coc.py`       | Официальный API Clash of Clans |
| `asyncpg`      | Работа с PostgreSQL            |
| `APScheduler`  | Планировщик задач              |
| `aiohttp`      | Веб-сервер для вебхуков        |

## Запуск
Бот работает автоматически 24/7 на удалённом сервере с постоянным обновлением данных.
